**Thundervim**
[Neovim](https://neovim.io/) LUA configuration, oriented for web developers (html, css, SCSS, javascript).

**DEPENDENCIES**

-Neovim 0.8+

**INSTALLATION**

```console
wget https://github.com/hdrdevs/thundervim/raw/main/install.sh && chmod +x install.sh && ./install.sh
```
